# Breast Cancer Classification using Machine Learning

This project focuses on breast cancer prediction using machine learning techniques and healthcare datasets.

## Techniques Used
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Logistic Regression
- Random Forest
- Model evaluation

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Goal
To explore predictive modeling approaches for healthcare-related classification problems using real-world biomedical data.
